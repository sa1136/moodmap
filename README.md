# MoodMap

MoodMap is a web application that provides personalized location recommendations based on a user's preferences, context, and current mood. It helps users find the perfect place to visit, whether it's a quiet cafe, an energetic gym, or a lively event.

## Tech Stack

**Frontend:** React.js, TypeScript, Tailwind CSS  
**Backend:** Node.js, Express.js, TypeScript  
**AI/ML:** Groq (Llama 3.3 70B)  
**APIs:** OpenStreetMap/Nominatim, Unsplash Source API  
**Location Services:** Browser Geolocation API
